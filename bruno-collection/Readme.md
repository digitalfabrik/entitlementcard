# Bruno workspace for Entitlementcard

This directory contains [Bruno](http://usebruno.com) collections for testing APIs of the entitlement card backend.

### Installation

See https://docs.usebruno.com/get-started/bruno-basics/download for options.

### Setup

- Copy `.env.example` to `.env` and edit to your needs
- Create a user import token in http://localhost:3000/project and set it in `.env` as `backend_user_import_token`
- Add the directory `/bruno-collection` as collection to your Bruno workspace.

### Usage

- Select the `dev` environment from the environment dropdown in the top right corner of Bruno.
- Select a request from the left sidebar.
- Run the request.
